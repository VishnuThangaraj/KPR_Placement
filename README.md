# KPR CAS Placement Class Training Materials

Welcome to the repository for the KPR College of Arts Science (KPR CAS) Placement Class. This repository contains all the notes, code snippets, and learning materials covered during the training sessions to prepare for technical interviews and data analyst roles.

## 📁 Repository Structure

The repository is organized into two primary learning tracks:

### 1. Problem Solving & DSA (March 2026)
The root directory contains daily text files (e.g., `02March2026.txt`, `03March2026.txt`, etc.) which act as a log of algorithmic problem-solving sessions.
- Focuses on **Data Structures and Algorithms (DSA)**.
- Contains solutions to popular **LeetCode** problems.
- Code snippets are primarily written in Java and JavaScript.
- Topics include Arrays, Linked Lists, Sliding Window, and other foundational interview topics.

### 2. Python - Data Analyst Track (May - June 2026)
Located in the `Python - Data Analyst` folder, this module is a comprehensive guide to Data Analysis using Python. It includes Jupyter Notebooks (`.ipynb`) detailing various stages of the data science lifecycle.

**Topics Covered:**
* **Basics & Core Libraries:** Python Basics (`29-May-2026`), NumPy (`30-May`), Pandas (`01-June`).
* **Data Processing:** File Handling (`02-June`), Data Cleaning (`03-June`).
* **Exploratory Data Analysis:** EDA concepts and practical implementations (`06-June`).
* **Visualization:** Building charts and graphs using Matplotlib/Seaborn (`03-June`, `04-June`).
* **Statistics:** Foundational and advanced statistical methods for data analysis (`04-June`, `05-June`).
* **Case Study:** A complete Data Science Life Cycle analysis featuring a Netflix Case Study.
* **Datasets:** Sample data used across the notebooks (found in the `Dataset` directory).

## 🚀 Getting Started

### For DSA Problem Solving
Simply open any of the `.txt` files in the root directory to review the LeetCode problem links and their corresponding optimized solutions.

### For Data Analyst Notebooks
To run the Jupyter notebooks locally, ensure you have Python installed along with the required libraries. 

1. Navigate to the Data Analyst folder:
   ```bash
   cd "Python - Data Analyst"
   ```
2. Create and activate a virtual environment (optional but recommended).
3. Install the standard data science stack (if you haven't already):
   ```bash
   pip install jupyter pandas numpy matplotlib seaborn scipy
   ```
4. Start the Jupyter Notebook server:
   ```bash
   jupyter notebook
   ```
